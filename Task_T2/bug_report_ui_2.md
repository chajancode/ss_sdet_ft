# Баг-репорт 2. UI. Некорректный вывод сообщения о ненайденных файлах.

## Описание:
При отсутствии в хранилище файла, удовлетворяющего поисковому запросу, возвращается ответ "Ничего не нашлось".
## Предусловия:
- Пользователь авторизован в сервисе
- Открыт интерфейс хранилища файлов
- Загружены файлы с именами: "Untitled.layout", "flant - 11.wav"
## Шаги воспроизведения
1. Кликнуть на значке поиска
2. Ввести имя файла "file-with-random-name.random"
3. Кликнуть кнопку "Найти" или нажать "Enter" на клавиатуре
## Ожидаемый результат
Вслучае, если нет файлов, удовлетворяющих запросу, получено сообщение "В вашем Диске не найдено файлов или папок по запросу <.......>"
## Фактический результат
Ответ: "Ничего не нашлось".  Присутствует возможность поиска в корзине и истории

## Окружение
Браузер: Google Chrome. Версия: 146.0.7680.153 (официальная сборка) (arm64)

## Серьёзность
Trivial

# Баг-репорт 3. UI. Зависание поиска при использовании спецсимвола.

## Описание:
Ввод в строку поиска символа "/" вызывает бесконечную анимацию поиска файлов.
## Предусловия:
- Пользователь авторизован в сервисе
- Открыт интерфейс хранилища файлов
- Загружены файлы с именами: "Untitled.layout", "flant - 11.wav"
## Шаги воспроизведения
1. Кликнуть на значке поиска
2. Ввести символ "/"
3. Кликнуть кнопку "Найти" или нажать "Enter" на клавиатуре
## Ожидаемый результат
Поиск вернул содержание корневой папки хранилища.
## Фактический результат
Выведено сообщение "Результаты поиска". Бесконечная анимация обработки поискового запроса.

## Окружение
Браузер: Google Chrome. Версия: 146.0.7680.153 (официальная сборка) (arm64)

## Серьёзность
Trivial

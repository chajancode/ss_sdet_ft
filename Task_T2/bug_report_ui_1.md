# Баг-репорт 1. UI. Не работает поиск файлов через строку поиска.

## Описание:
При наличии в хранилище файла, удовлетворяющего поисковому запросу, возвращается ответ "Ничего не нашлось".
## Предусловия:
- Пользователь авторизован в сервисе
- Открыт интерфейс хранилища файлов
- Загружены файлы с именами: "Untitled.layout", "flant - 11.wav"
## Шаги воспроизведения
1. Кликнуть на значке поиска
2. Ввести имя файла "flant - 11.wav"
3. Кликнуть кнопку "Найти" или нажать "Enter" на клавиатуре
## Ожидаемый результат
Список файлов и папок, содержащих в своем названии текст запроса.
## Фактический результат
Ответ: "Ничего не нашлось".  Присутствует возможность поиска в корзине и истории

## Окружение
Браузер: Google Chrome. Версия: 146.0.7680.153 (официальная сборка) (arm64)

## Серьёзность
Major
